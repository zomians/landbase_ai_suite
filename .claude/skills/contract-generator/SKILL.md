# 業務委託契約書HTML生成スキル

月額固定報酬とレベニューシェア条項を含む業務委託契約書をHTMLで自動生成します。ブラウザの印刷機能でPDF化する運用です。

## 使い方

```
/contract-generator <クライアント名> <契約開始日> [オプション]
```

**引数**:
- `<クライアント名>` (必須): 委託者（甲）の正式名称（例: 株式会社サンプル）
- `<契約開始日>` (必須): 契約開始日（例: 2026-04-01）
- 報酬条件・その他詳細はインタラクティブに確認する

## 実行手順

### Step 1: 契約情報の収集

ユーザーから以下の情報を収集する。不足があれば質問して確認する。

#### 甲（委託者）情報

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 会社名（甲） | Yes | 委託者の正式名称 |
| 代表者名（甲） | Yes | 代表者または担当者名 |
| 住所（甲） | Yes | 委託者の住所 |
| 契約開始日 | Yes | 契約の効力発生日 |

#### 月額報酬

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 月額報酬額 | Yes | 税抜き金額（例: 200,000円） |
| 支払期日 | Yes | 例: 毎月末日、翌月25日 |
| 支払方法 | Yes | 例: 銀行振込（振込先口座を記載） |

#### レベニューシェア（オプション）

| 項目 | 必須 | 説明 |
|------|:----:|------|
| レベニューシェアの有無 | Yes | あり / なし |
| 対象売上の定義 | No | 「総売上」or「純売上（返金・手数料控除後）」 |
| シェア率 | No | 例: 売上の 10% |
| 計算・支払サイクル | No | 「月次」or「四半期」 |
| 最低保証額 | No | 例: 月額 50,000円（省略可） |
| 上限額 | No | 例: 月額 500,000円（省略可） |

### Step 2: 契約書番号の採番

契約書番号は `CTR-YYYYMM-NNNN` 形式で採番する。

1. `docs/templates/contract/` ディレクトリ内の既存HTMLファイルを Glob で検索する
2. ファイル名から最新の契約書番号を特定する
3. 同月内で連番をインクリメントする（初回は `0001`）

例: `CTR-202604-0001`, `CTR-202604-0002`

### Step 3: HTMLファイルの生成

以下の構成・デザインで業務委託契約書HTMLを生成する。

#### 乙（受託者）固定情報

```
株式会社AI.LandBase
代表取締役　末永 壽蔵
〒905-0412 沖縄県国頭郡今帰仁村湧川852-2
```

#### ページ構成

**ページ1: 本文**（`docs/templates/contract/CTR-YYYYMM-NNNN.html` の `.page` 第1ブロック）

- タイトル「業 務 委 託 基 本 契 約 書」（字間スペース・中央揃え）
- 締結日記入欄（右揃え・下線）
- 当事者情報ボックス（枠線あり・甲は空欄・乙は固定）
- 前文
- 第1〜10条の本文
- 末文・署名欄（甲乙並列ボックス・印鑑枠）

**ページ2: 別紙**（`.page.page-break` 第2ブロック）

- 右上に「業務委託基本契約書　別紙」と表示
- タイトル「別 紙　報 酬 条 件 表」
- 報酬条件表（縦2列テーブル: 項目 / 条件詳細）
- 甲乙署名欄

#### 生成する条項（全10条）

以下の条項をすべて含む契約書本文を生成する。

**第１条（目的）**
- 本契約は、甲が乙に対して業務を委託し、乙がこれを受託するにあたっての基本的な条件を定めることを目的とする。

**第２条（委託業務の内容）**
- 乙が甲に提供する業務は①システム開発支援業務・②経営助言業務の2種とする
- 各業務の詳細は別途書面または電磁的方法で定める

**第３条（報酬）**（3項構成）
- 第1項: 甲は別紙「報酬条件表」に定める報酬を支払う
- 第2項: 各報酬に係る消費税は甲が別途負担する（外税）
- 第3項: 報酬条件の変更は書面合意のうえ別紙を改訂することにより行う

**第４条（支払条件）**（3項構成）
- 第1項: 乙は毎月末日締めで請求書を送付する
- 第2項: 甲は翌月末日までに振込む。振込手数料は甲負担
- 第3項: 遅延の場合、年率14.6%の遅延損害金

**第５条（契約期間）**（3項構成）
- 第1項: 開始日から1か月間
- 第2項: 期間満了1か月前までに申し入れなき場合は同条件で1か月自動更新（以後同様）
- 第3項: 1か月前書面通知で契約期間満了時に解約可

**第６条（秘密保持）**（2項構成）
- 第1項: 技術上・営業上・財務上の情報を第三者に開示・漏洩してはならない
- 第2項: 義務は契約終了後2年間存続

**第７条（知的財産権）**（3項構成）
- 第1項: 乙固有技術を含む成果物の知的財産権は乙帰属。甲は乙承諾なく転載・譲渡・改変不可
- 第2項: 乙固有技術を含まない成果物は、対価完済を条件に甲帰属
- 第3項: 乙が本業務以前から保有する技術・ノウハウは乙帰属

**第８条（損害賠償）**（3項構成）
- 第1項: 善良な管理者の注意義務・賠償責任
- 第2項: 賠償範囲は直接かつ通常の損害に限定（逸失利益・間接・特別損害除く）。上限は損害発生月前1か月の報酬総額
- 第3項: 故意または重大過失の場合は上限制限を適用しない

**第９条（反社会的勢力の排除）**
- 両当事者は現在および将来にわたり反社会的勢力に該当しないことを表明・保証する
- 違反が判明した場合、相手方は催告なく解除できる

**第１０条（協議・合意管轄）**（2項構成）
- 第1項: 定めのない事項・疑義は誠実に協議する
- 第2項: 紛争は那覇地方裁判所または東京地方裁判所を第一審の専属的合意管轄裁判所とする

#### 署名欄・別紙

**ページ1 署名欄**（末文の後）

```
本契約の成立を証するため、本書２通を作成し、甲乙各１通を保有する。
[罫線]
署　名　欄

[甲ボックス]                    [乙ボックス（乙情報固定）]
所在地：＿＿＿               所在地：〒905-0412 沖縄県国頭郡今帰仁村湧川852-2
法人名：＿＿＿               法人名：株式会社 AI.LandBase
代表者：＿＿＿ ㊞            代表者：代表取締役　末永 壽蔵 ㊞
[印鑑枠（円形）]              [印鑑枠（円形）]
```

**ページ2 別紙　報酬条件表**

| 項目 | 条件詳細 |
|------|---------|
| ①システム開発支援業務報酬（月額固定） | 月額\_\_\_円（税抜） |
| ②経営助言業務報酬（レベニューシェア） | 対象売上の定義\_\_\_ ／ シェア率\_\_\_% ／ 計算・支払サイクル\_\_\_ |
| 消費税の取扱い | ①②ともに消費税は甲が別途負担する（外税） |
| 適用開始日 | \_\_\_年\_\_\_月\_\_\_日以降の請求分から適用 |
| 特記事項 | 書面合意で改訂可。改訂後の別紙は本契約と同等の効力を有し旧別紙と差し替え |

別紙末尾に甲乙署名欄（所在地・法人名・代表者・印鑑枠）を配置する。

### Step 4: ファイルの保存

1. 生成したHTMLを `docs/templates/contract/CTR-YYYYMM-NNNN.html` として Write ツールで保存する
2. ユーザーに以下を報告する:
   - 出力ファイルパス
   - 契約書番号
   - 委託者（甲）
   - 契約開始日
   - 月額報酬額（税抜）
   - レベニューシェア条件（設定がある場合）
3. 「ブラウザで開いて印刷/PDF保存してください」と案内する
