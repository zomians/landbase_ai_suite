## 2. Instagram マーケティング戦略 2025

### 2.1 エグゼクティブサマリー

**2025年のInstagramマーケティング5大トレンド**:

1. **ショート動画（Reels）の絶対優位**: 視聴完了率と送信数が最重要指標
2. **ソーシャルコマースの本格化**: Instagram Shop + 決済統合
3. **冷凍食品のプレミアム化**: "便利"から"ギフト品質"へ
4. **AI生成コンテンツとの共存**: オリジナリティが差別化要因
5. **週1-2回の戦略的投稿**: 量より質、エンゲージメント重視

**Shrimp Shells への推奨戦略**:

- **Reels中心**: 投稿の80%をReelsに
- **週1-2回投稿**: 高品質コンテンツに集中
- **視聴完了率最優先**: 30-90秒の最適尺
- **Instagram Shopping統合**: Solidus EC との連携
- **半自動化フロー**: Claude API で下書き生成 → 人間承認 → 投稿

### 2.2 Instagram アルゴリズム完全解説（2025年版）

#### 2.2.1 ランキングシグナル TOP 3

Meta（Instagram運営）が公式発表している2025年の最重要ランキング要素：

| ランキングシグナル | 重要度 | 対象 | 説明 |
|------------------|--------|------|------|
| **Average Watch Time（平均視聴時間）** | ⭐️⭐️⭐️⭐️⭐️ | 全ユーザー | 最後まで見られた動画を優先表示 |
| **Likes Per Reach（リーチあたりのいいね数）** | ⭐️⭐️⭐️⭐️ | フォロワー | 既存フォロワーへのリーチに影響 |
| **Sends Per Reach（リーチあたりのシェア数）** | ⭐️⭐️⭐️⭐️⭐️ | 新規ユーザー | **新規リーチ拡大の最重要指標** |

**重要な変化**:
- **2024年以前**: いいね数が全体的に重要
- **2025年**: フォロワー向けは「いいね」、新規向けは「シェア」と明確に分離

**Shrimp Shells 戦略**:
- 新規顧客獲得が目標 → **シェア数を最優先KPI**に設定
- "シェアしたくなる"コンテンツ設計が必須

```ruby
# app/services/ai_sns_content_generator.rb

class AiSnsContentGenerator
  def generate_reels_script(pillar:, product:)
    brand_story = <<~STORY
      Shrimp Shells ブランドストーリー:
      - 横浜のプロ料理人が、ハワイのグルメな友人と食べ歩き

      - "ハワイのホームパーティ体験"をオンラインで届ける
      - 港町・横浜とハワイのつながり
    STORY

    prompt = <<~PROMPT
      あなたはShrimp ShellsのInstagram Reelsコンテンツクリエイターです。

      #{brand_story}

      【コンテンツピラー】
      #{pillar}

      【商品情報】
      #{product.to_json}

      【2025年Instagram最適化要件】
      - 動画尺: 60-90秒（ホームパーティ体験）、30-60秒（その他）
      - 最初の3秒で強烈なフック
      - シェアしたくなる要素（"ハワイ好きな友達に教えたい"）
      - 視聴完了率を最大化
      - トレンド音源を想定

      【出力形式】
      JSON形式:
      {
        "pillar": "ピラー名",
        "title": "投稿タイトル",
        "hook": "最初の3秒の内容",
        "script": [
          {
            "time": "0-3秒",
            "visual": "映像の内容",
            "text_overlay": "画面テキスト",
            "narration": "ナレーション（あれば）"
          },
          ...
        ],
        "caption": "投稿文（150文字、ブランドストーリーを感じさせる）",
        "hashtags": ["タグ1", "タグ2", ...],  // 15-25個、3層構造
        "cta": "行動喚起文",
        "music_suggestion": "推奨音源の雰囲気",
        "expected_engagement": {
          "watch_time": "予想視聴完了率",
          "share_potential": "シェア可能性（高/中/低）",
          "target_audience": "ターゲット層"
        }
      }
    PROMPT

    response = call_claude_api(prompt)
    JSON.parse(response)
  end

  private

  def call_claude_api(prompt)
    response = HTTParty.post(
      'https://api.anthropic.com/v1/messages',
      headers: {
        'x-api-key' => ENV['ANTHROPIC_API_KEY'],
        'anthropic-version' => '2023-06-01',
        'content-type' => 'application/json'
      },
      body: {
        model: 'claude-3-5-sonnet-20250514',
        max_tokens: 3000,
        messages: [{ role: 'user', content: prompt }]
      }.to_json
    )

    response['content'][0]['text']
  end
end
```
